Open Banking Insights — Fintech Analytics Project**

This project analyses synthetic open banking transaction data to extract meaningful financial insights. It demonstrates practical skills used in modern fintech products, including cashflow analysis, subscription tracking, spending categorisation, and anomaly detection.
The goal is to showcase hands‑on fintech analytics, security awareness, and data‑driven product thinking.

Tech Stack
- Python  
- Pandas  
- Matplotlib  
- Jupyter Notebook  

Features
 Monthly Cashflow Analysis
Tracks income vs spending across months to reveal financial trends and stability.

Spending by Category
Breaks down spending into categories such as groceries, rent, utilities, and entertainment.

Subscription Tracking
Identifies recurring merchants (e.g., Netflix, Spotify, BT) and calculates monthly subscription costs.

Anomaly Detection (Security Angle)
Flags unusually large transactions using a quantile‑based threshold.  
This mirrors early‑stage fraud detection logic used in fintech security systems.

Visualisations Included
- Monthly cashflow line chart  
- Category spending bar chart  
- Subscription spending trend  
- Anomaly detection output
